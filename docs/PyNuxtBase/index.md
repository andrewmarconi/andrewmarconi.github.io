# PyNuxtBase

An opinionated, modern, full-stack web application foundation built with **Nuxt 4**, **FastAPI**, and **Django ORM**.

![PyNuxtBase Logo](/img/pynuxtbase-logo.png)

## Overview

PyNuxtBase provides a production-ready bootstrapping foundation for modern web applications with a unique hybrid backend architecture that combines the best of Django and FastAPI.

### Key Features

- ⚡ **Modern Stack**: Nuxt 4, Vue 3, Python 3.12, FastAPI, Django
- 🔒 **Secure by Default**: JWT authentication with refresh tokens, UUID primary keys
- 📘 **Type Safety**: Strictest TypeScript + Python mypy configuration
- 🎨 **Design System**: Comprehensive UI component library with Tailwind CSS 4
- 🐳 **Docker Ready**: Complete local development environment
- 🔄 **Hybrid Backend**: Django ORM + FastAPI in a single process

## Technology Stack

### Frontend
- **Nuxt 4** - Server-side rendering and static generation
- **Vue 3** - Progressive JavaScript framework
- **TypeScript** - Strictest type safety
- **Tailwind CSS 4** - Utility-first CSS (CSS-only configuration)
- **Pinia** - State management
- **Zod** - Runtime validation

### Backend
- **Python 3.12** - Latest Python version
- **Django 6.0** - ORM, admin interface, and migrations
- **FastAPI 0.115** - Modern async API framework
- **Uvicorn** - Lightning-fast ASGI server
- **Pydantic 2.x** - Data validation using Python type annotations

### Infrastructure
- **PostgreSQL 17** - Advanced relational database
- **Docker Compose** - Multi-container development environment
- **Caddy** - Modern reverse proxy
- **Mailpit** - Email testing in development
